# 🚀 Internship Finder (ML-Based Recommendation System)

A smart internship recommendation system that analyzes user resumes and suggests the most relevant internships based on skills and profile using Machine Learning and NLP.

---

## 📌 Overview

This project allows users to upload their resumes, extracts key skills using NLP techniques, and recommends suitable internships by matching them with available opportunities. It aims to simplify the internship search process by providing personalized suggestions.

---

## ✨ Features

* 📄 Resume Upload (PDF/Text)
* 🧠 Skill Extraction using NLP
* 🤖 ML-based Internship Recommendation
* 📊 Matching based on similarity scores
* 🌐 Interactive UI using Streamlit
* 💾 User data storage using SQLite

---

## 🛠️ Tech Stack

* **Frontend:** Streamlit
* **Backend:** Python
* **Machine Learning:** NLP, Text Processing, Similarity Matching
* **Database:** SQLite

---

## ⚙️ How It Works

1. User uploads their resume
2. NLP processes the resume and extracts skills
3. ML model compares extracted skills with internship dataset
4. System recommends best matching internships
